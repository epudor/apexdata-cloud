"AI" covers two very different capabilities. Knowing which one your problem needs prevents a lot of wasted effort.

## Side by side

| Dimension | Predictive AI | Generative AI |
|---|---|---|
| Core job | Forecast / classify | Create content |
| Examples | Churn, demand, risk, propensity | Text, code, images, summaries, answers |
| Output | A number or label | New content |
| Data | Historical, labeled | Large pretraining + your knowledge via [RAG](/glossary/retrieval-augmented-generation/) |
| Typical service | [Machine learning](/services/machine-learning-consulting/) | [Generative AI](/services/generative-ai-consulting/) |

## When to use which

- Need to **forecast, score, or classify**? → **Predictive AI** (machine learning).
- Need to **generate, summarize, or answer**? → **Generative AI**.
- Many high-value solutions use **both** — e.g., predict which customers will churn, then generate a personalized retention offer.
